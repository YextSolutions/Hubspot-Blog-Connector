##### Hubspot-Blog-Connector

# Product Description & Purpose

Pull in your blogs with Yext's new Hubspot Data Connector! Hubspot CMS Hub integrates all your website tools in one place. With the addition of the Hubspot CMS Hub data connector, you can retrieve all of your blogs into your Knowledge Graph to be used in Yext products.

The Hubspot Blog Connector pulls in Blog data such as name, body, author, and post date into Yext. This app uses the built-in Help Article entity type and creates two custom fields:

- Author Name
- Featured Image

# Requirements

### To use this template you will need to have the following before you install:

- a Hubspot developer account 
- your Hubspot API Key

### Follow the step-by-step instructions below to install the Hubspot Blog Connector app.

# How to Install

If you are an existing Yext customer, you can install the Hubspot Blog Connector here <https://www.yext.com/s/me/apps/66646>

If you are currently using a Yext sandbox account, you can install the Hubspot Blog Connector here <https://sandbox.yext.com/s/me/apps/135109>.

If you are not an existing customer, but interested in learning more, try out a free trial here for a production account, or sign up for Hitchhikers and get started with a sandbox account, here <https://hitchhikers.yext.com/>. 

### To install the Hubspot Blog Connector:

1. Log in to your Yext account
2. Navigate to the **Apps > Directory** tab in the platform (or use the link provided above).
3. Search for the Hubspot Blog Connector app and click **Install**.
4. Accept the following changes to your account by clicking **Next** and authorize Yext APIs.
5. Provide your Hubspot API Key.

Wait for the connectors to run and see all of your newly created Blog entities in the **Knowledge Graph** tab in the platform!
